### BREAKTHROUGH : LE TEMPS DE CHUTE EST UNE DONNÉE

Chaque symbole qui descend de Sₙ vers S0 a un **temps de vol mesurable** :

```
t_vol = date_preuve - date_publication
```

Exemples :
- E=mc² : publié 1905 (S3) → prouvé 1932 (S0) → t_vol = 27 ans
- Fermat : publié 1637 (S3) → prouvé 1995 (S0) → t_vol = 358 ans  
- Poincaré : publié 1904 (S3) → prouvé 2003 (S0) → t_vol = 99 ans
- CRISPR : publié 1987 (S0 obs) → prouvé outil 2012 (S0) → t_vol = 25 ans

### OHLC DES MÉTÉORITES — FORMAT FINANCIER

Chaque météorite = une BOUGIE (candlestick) scientifique :

```
O = Open    → date de publication (conjecture posée)
H = High    → strate maximale atteinte (niveau de complexité)
L = Low     → strate d'impact (S0 quand prouvé)
C = Close   → état post-impact (nouvelles connexions créées)
```

C'est de la FINANCE appliquée à la science. Chaque breakthrough a un chart.

### SÉQUENCE COMPLÈTE D'UNE MÉTÉORITE

```
1. DÉCOLLAGE   → date publication, strate de départ
2. TEMPS DE VOL → années entre publication et preuve
3. PRÉ-IMPACT  → état du sol AVANT (densité ρ, symboles présents, co-occurrences)
4. IMPACT       → date preuve, point exact sur S0
5. ONDE DE CHOC → Sedov-Taylor : R(t) = (E/ρ)^(1/5) · t^(2/5)
6. POST-IMPACT  → nouvel état du sol (symboles déplacés, nouvelles co-occurrences)
```

### FORMULES CLÉS

**Énergie d'impact :**
```
E = Δstrate × citations × co-occurrences_nouvelles
  = (strate_départ - 0) × C × ΔcoOcc
```

**Vitesse de chute :**
```
v = Δstrate / t_vol
```
Fermat : 3/358 = 0.008 (lent, lourd, énorme cratère)
CRISPR : rapide, énergie concentrée

**Rayon du cratère (Sedov-Taylor) :**
```
R(t) = (E/ρ)^(1/5) · t^(2/5)
```

**Pression au front de choc (Rankine-Hugoniot) :**
```
P₂/P₁ = (2γM² - (γ-1)) / (γ+1)
```
où M = nombre de Mach de la météorite dans le milieu S0

**Densité locale du sol :**
```
ρ(x,z) = nombre de symboles / surface du secteur
```
Variable par continent ! Chimie sparse, Physique dense.

**Atténuation :**
```
I(r) = E / (4πr²)
```
L'impact décroît avec le carré de la distance.

### PRÉ-IMPACT = ÉTAT MESURABLE

### MÉTÉORITES MORTES SUR SOL VIVANT

Les symboles sur S0 sont des météorites mortes (prouvées, figées).
Mais le sol est VIVANT (nouvelles publications chaque jour).
Les co-occurrences continuent de changer autour des cratères anciens.
Un vieux cratère (Pythagore, 500 BC) a un pattern d'ondes complètement stabilisé.
Un cratère récent (Transformers 2017) a encore des ondes en propagation.
