## 5 PATTERNS IDENTIFIÉS SUR 7 TESTS RÉELS

### PATTERN 1 — PONT INTER-DOMAINES
**Signature** : Deux clusters massifs, connexion <3%, un paper/équipe crée le pont → explosion
**Exemples** : Fermat (EC↔MF, 1.8%→pont Wiles), CRISPR (triple 5→134 en 1 an), Deep Learning (DL+CNN vs CV, x675)
**Ce que le moteur voit** : Trou structural entre clusters denses
**Ce que ça prédit** : La percée viendra d'une CONNEXION, pas de plus de travail dans chaque cluster
**Indicateur** : co-occurrence < 3% entre clusters de >100 papers/an chacun

### PATTERN 2 — ZONE DENSE INTRA-DOMAINE  
**Signature** : Un seul cluster, connexion >50%, résolu par accumulation collective
**Exemple** : Classification des groupes finis (CFSG↔GT = 77-100%)
**Ce que le moteur voit** : Réseau dense, pas de trou
**Ce que ça prédit** : Pas besoin de pont, besoin de MAIN D'ŒUVRE
**Indicateur** : co-occurrence > 50%, un seul domaine dominant

### PATTERN 3 — THÉORIE PRÊTE, OUTIL MANQUANT
**Signature** : Connexion théorique dense, connexion expérimentale quasi-nulle
**Exemple** : Higgs (H↔SM dense dès 1985, H↔LHC = 0 jusqu'au LHC construit)
**Ce que le moteur voit** : Asymétrie théorie/expérience
**Ce que ça prédit** : La percée viendra d'un OUTIL, pas d'une idée
**Indicateur** : concept_A↔concept_B > 10% MAIS concept_A↔instrument < 1%

### PATTERN 4 — TROU ENCORE OUVERT
**Signature** : Deux clusters massifs, connexion <2%, stable sur 30+ ans
**Exemple** : Dark Matter ↔ Quantum Gravity (1.5% en 2024, stable depuis 1990)
**Ce que le moteur voit** : Trou structural persistant
**Ce que ça prédit** : PROBLÈME NON RÉSOLU — la percée nécessite un pont qui n'existe pas encore
**Indicateur** : co-occurrence < 2% stable sur >20 ans entre clusters de >1000 papers/an

### PATTERN 5 — CROISSANCE SANS PONT (promesse non tenue)
**Signature** : Un cluster explose (x100) mais le pont vers l'application reste faible
**Exemple** : Graphène (108→25000 papers/an, mais GR↔Semiconductor reste <2%)
**Ce que le moteur voit** : Cluster en hyper-croissance, pont stagnant
**Ce que ça prédit** : Le domaine est ACTIF mais la connexion promise ne se fait pas
**Indicateur** : cluster growth >x10 ET pont co-occurrence stagnant <2%
